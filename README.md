##VPNC
####Setup:
The version of Cisco Client must be 3.1.05152 or newer.

Create file ``.uap`` with

    VPN URL
    USERNAME
    PASSWORD

``Home-brew``

 - terminal-notifier 

====
###Bash
run vpnc.sh to connect

====
###Ruby
- run ``rbvpnc`` to connect
- run ``svpnc`` to check state
- run ``dvpnc`` to disconnect

====
